Detailed Description

3D renderer implementation for the GUI, handling OpenGL rendering and resource management.

This class is responsible for initializing, updating, rendering, and shutting down the 3D rendering context. It manages OpenGL resources, shaders, camera, terrain, water, and player rendering.
