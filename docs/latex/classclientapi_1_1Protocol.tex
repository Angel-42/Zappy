\doxysubsection{Detailed Description}
Static utility class containing all Zappy network protocol constants and helper functions. 

This class centralizes all protocol-\/related constants and provides utilities for\+:
\begin{DoxyItemize}
\item Command validation and parsing
\item Message formatting according to Zappy specification
\item Time limit management for different commands
\item \doxylink{classclientapi_1_1Protocol}{Protocol} compliance checking
\end{DoxyItemize}

The \doxylink{classclientapi_1_1Protocol}{Protocol} class implements the exact specifications from the Zappy project requirements\+:
\begin{DoxyItemize}
\item AI client commands with proper time limits
\item Standard response messages (ok, ko, dead, etc.)
\item Command buffer limits (max 10 commands)
\item Authentication flow (WELCOME -\/\texorpdfstring{$>$}{>} team name -\/\texorpdfstring{$>$}{>} game info)
\end{DoxyItemize}

\begin{DoxyNote}{Note}
All members are static -\/ this is a utility class, not meant to be instantiated 
\end{DoxyNote}
